{% blocktrans %}Types of Crime{% endblocktrans %}

{% blocktrans %}Theft{% endblocktrans %}

{% blocktrans %}Unlawfully taking someone else's property without permission.{% endblocktrans %}

{% blocktrans %}Violence{% endblocktrans %}

{% blocktrans %}Physical harm or threats to people, including assault and abuse.{% endblocktrans %}

{% blocktrans %}Cybercrime{% endblocktrans %}

{% blocktrans %}Illegal activities conducted online like hacking, fraud, or identity theft.{% endblocktrans %}

{% blocktrans %}How to Stay Safe & Prevent Crime{% endblocktrans %}

{% blocktrans %}Personal Safety{% endblocktrans %}

{% blocktrans %}Always be aware of your surroundings, avoid isolated areas, and travel with trusted companions.{% endblocktrans %}

{% blocktrans %}Home & Property Safety{% endblocktrans %}

{% blocktrans %}Lock doors and windows, install alarms or CCTV, and avoid sharing personal info publicly.{% endblocktrans %}

{% blocktrans %}Online Safety{% endblocktrans %}

{% blocktrans %}Use strong passwords, avoid clicking unknown links, and never share sensitive information online.{% endblocktrans %}

{% blocktrans %}Community Awareness{% endblocktrans %}

{% blocktrans %}Participate in neighborhood watch programs and report suspicious activities immediately.{% endblocktrans %}
